## Format Requirements

- File format: WebP (preferred) or PNG
- Transparent background recommended
- Dimensions: Approximately 200-300px wide
- Grayscale/monochrome versions work best for the showcase

## Usage

The landing page will automatically display these logos in the "Trusted Partners" section. The logos are shown in grayscale by default and become full color on hover.
