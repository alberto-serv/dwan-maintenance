# Dwan Elevator Landing Page

A modern, conversion-optimized landing page for Dwan Elevator Company's commercial elevator maintenance services.

## Features

- **First Year Free Offer** - Prominent hero section highlighting the unique offer
- **Mobile-Optimized** - Fully responsive design for all devices
- **Conversion-Focused** - Strategic CTAs and lead capture form
- **Brand Compliance** - Follows Dwan Elevator branding guidelines
- **Built with Modern Stack**:
  - Next.js 14
  - TypeScript
  - Tailwind CSS
  - shadcn/ui components

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

To create a production build:

```bash
npm run build
npm start
```

## Deployment

This site can be deployed to Vercel, Netlify, or any platform that supports Next.js.

## Branding

- Primary Color: #EFBF04 (Gold)
- Background: #FFFFFF (White)
- Text: #000000 (Black)
- Link Color: #CC3366 (Pink)
- Fonts: Montserrat (headings), Poppins (body)
- Border Radius: 3px
- Base Unit: 4px
